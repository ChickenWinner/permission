# permission

#### 介绍
基于RBAC数据库模型下的权限管理系统
1. 可实现用户、角色模块、角色、权限模块、权限的CRUD
2. 实现登录的过滤和权限的拦截

#### 使用技术
1. springboot、mybatis搭建后台
2. maven进行项目管理
3. redis实现缓存功能
4. 前台html+css+jQuery+ztree渲染界面

#### 实现功能
1. 用户、角色、用户权限的增加、删除、修改、更新（CRUD）
2. 用户与角色、角色与权限的关联关系
3. 利用 redis 将用户信息、权限信息缓存，提高系统运行速度
4. 权限的拦截、登录的过
